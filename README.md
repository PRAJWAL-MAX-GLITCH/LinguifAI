## ✨ Features
- **Real-time Translation**: Seamlessly translate text across multiple languages with high accuracy.
- **Modern Architecture**: Clean separation between a robust Backend and a dynamic Frontend.
- **Containerized deployment**: Easy local setup with Docker and `docker-compose`.

## 🚀 Quick Start

### Prerequisites
- [Docker](https://www.docker.com/) and Docker Compose installed on your system.

### Installation
1. **Clone the repository**
   ```bash
   git clone https://github.com/PRAJWAL-MAX-GLITCH/LinguifAI.git
   cd LinguifAI
   ```

2. **Start the application**
   ```bash
   docker-compose up -d
   ```

3. **Access the application**
   - Frontend: `http://localhost:3000` (or the configured port)
   - Backend API: `http://localhost:8000` (or the configured port)

## 📁 Project Structure
- `/frontend`: Modern web interface.
- `/backend`: Scalable translation service API.
- `docker-compose.yml`: Configuration for running the full stack easily.
